# Safealert: A Smart IoT based Early Warning and Emergency Response System for Hazard Preparedness and Disaster Resilience

Safealert is a new early warning and emergency response system based on the Internet of Things (IoT). It aims is to make communities more resilient and better prepared for disaster. The systems use a network of connected sensors to continuously monitor critical environmental parameters such as temperature, water levels, rain gauge levels at all time.
